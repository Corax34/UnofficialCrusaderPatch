Field Value Type Range of Acceptable Values Description
CriticalPopularity Integer 0 to 10000 If popularity drops below this value the AI sells more stuff than usual to get money. 10000 is 100 popularity
LowestPopularity Integer 0 to 10000 If popularity drops below this value the AI sets taxes (in steps) to TaxesMin. 10000 is 100 popularity
HighestPopularity Integer 0 to 10000 If popularity is above this value the AI sets taxes back up until a stable 0 to -3 current popularity modifier is reached. 10000 is 100 popularity
