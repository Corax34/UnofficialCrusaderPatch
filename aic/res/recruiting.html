Field Value Type Range of Acceptable Values Description
RecruitInterval Integer 0 to 231 - 1 Default interval at which the AI recruits troops
RecruitIntervalWeak Integer 0 to 231 - 1 Recruiting interval for 'weak' state which sets in if (ie. AI has troops <= 8)
RecruitIntervalStrong Integer 0 to 231 - 1 Recruiting interval for 'strong' state which sets in if (ie. AI has troops >= 40)
RecruitProbDefensive
Default
Strong
Weak
Integer 0 to 100 Probability that the AI changes its recruitment type to defensive troops in each state. Note: These values must sum to 100. Sortie units are always recruited first. The AI always starts the game with defensive recruitment state until the AIV is completely built
RecruitProbRaid
Default
Strong
Weak
Integer 0 to 100 Probability that the AI changes its recruitment type to raiding troops in each state Note: These values must sum to 100. Sortie units are always recruited first. The AI always starts the game with defensive recruitment state until the AIV is completely built
RecruitProbAttacking
Default
Strong
Weak
Integer 0 to 100 Probability that the AI changes its recruitment type to attacking troops in each state Note: The three values must sum to 100. Sortie units are always recruited first. The AI always starts the game with defensive recruitment state until the AIV is completely built
