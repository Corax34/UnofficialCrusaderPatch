Field Value Type Range of Acceptable Values Description
DefTotal Integer 0 to 231 - 1 The total count of all defensive units (wall defense + patrols) if the AI is in its 'strong' state
DefWalls Integer 0 to 231 - 1 Total number of troops for wall defense
DefWallPatrolRallyTime Integer 0 to 231 - 1 The amount of time for castle patrols set up in the AIV to move from one rally point to the next. (Note: Currently only spearmen, horse archers and macemen will patrol)
DefWallPatrolGroups Integer 0 to 231 - 1 The amount of concurrent existing patrol groups in the AIV setup defined as castle patrols per unit type
OuterPatrolGroupsCount Integer 0 to 231 - 1 The number of splits for outer defensive patrols.
OuterPatrolGroupsMove Bool "True" or "False" Controls whether the patrols stay at one place (quarry) or move around.
OuterPatrolRallyDelay Integer 0 to 231 - 1 The delay after which the AI sends out patrols to defend the outer economy. 4 is approximately one month
SortieUnits
Ranged Unit Unit Type of ranged units that go to the last attacked farm or building
RangedMin Integer 0 to 231 - 1 These units are only sent out if at least this number of them have already been recruited. More than the set amount can be recruited if the AI loses resource building workers frequently.
Melee Unit Unit Type of melee units recruited to attack enemy units shooting at the AI's buildings or workers.
MeleeMin Integer 0 to 231 - 1 Sortie units are only sent out if this amount of them has already been recruited.
DefDiggingUnitMax Integer 0 to 231 - 1 Number of units recruited to dig AIs own moat.
DefDiggingUnit DiggingUnit DiggingUnit Type of unit used to dig the AIs own moat.
